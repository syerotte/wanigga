# Free Mobile Website

This is a simple, responsive website starter that works well on iPhone and can be published for free.

## Quick preview
Open [index.html](index.html) in your browser, or run a local server from this folder.

## Free publishing options
1. Create a GitHub repository.
2. Upload these files.
3. Open Settings > Pages.
4. Choose Deploy from branch: main / root.
5. Your site will get a free public URL.

You can also use Netlify or Vercel for free hosting.
